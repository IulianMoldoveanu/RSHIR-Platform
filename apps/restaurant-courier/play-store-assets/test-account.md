## Instructions pentru Google Play reviewer

Text de copiat în Play Console -> App access -> Instructions:

```
HIR Curier requires courier credentials linked to a HIR partner fleet. New users cannot self-register from the app.

To test the app:

1. Open the app and tap "Login"
2. Enter the test email and password provided in the App access section
3. After login, you'll see the courier dashboard
4. Tap "Start shift" — the app will request location permission (please allow "while using the app" or "all the time" for full feature testing)
5. You'll see 3 mock orders pre-seeded for testing
6. Tap any order to view details
7. Tap "Accept" to simulate accepting a delivery
8. The map and ETA features will activate

Notes:
- The test account is pre-assigned to a DEMO fleet with 3 mock orders
- Location data is NOT shared outside the demo fleet
- Push notifications require a real Firebase environment (test account is configured for FCM)
- Test account can be reset between review cycles — contact support@hirforyou.ro if you need a reset

For any access issues, contact: curier@hirforyou.ro (reply within 24h).
```
