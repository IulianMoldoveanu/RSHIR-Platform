# Text Overlays — exact timing și conținut pentru video

---

## Convention generală

| Property | Valoare standard |
|---|---|
| **Font** | Inter Bold (sau Open Sans Bold) |
| **Background** | Rounded rectangle, opacity 60% black |
| **Padding** | 12px pe toate laturile |
| **Animation in** | Fade in 0.3 sec |
| **Animation out** | Fade out 0.4 sec |
| **Border radius** | 8px |

## Reguli generale

1. **NU mai mult de 1 overlay simultan** pe ecran (excepție: 5.2 cu comparație side-by-side, OK)
2. **Hold minim 3 sec** ca să poată viewer-ul citi (5 sec ideal pentru cele lungi)
3. **Hold maxim 8 sec** ca să nu devină distracting
4. **Animation in/out diferă în function de importanță:**
   - **Fade simple** pentru context
   - **Scale-up "pop"** pentru momente WOW (Hepi, 15.400 lei)
   - **Slide-in** pentru context expansion
5. **Color discipline:**
   - Verde (#22C55E) → numere pozitive, win-uri pentru patron
   - Roșu (#DC2626) → pain points, ce pierde patronul
   - Portocaliu (#FF6B35) → brand HIR (CTA, accent)
   - Alb pe negru → text contextual neutru
6. **Font discipline:** **un singur font** în întreg video (Inter Bold sau Open Sans Bold). NU mix fonts.

---

## Common mistakes de evitat

| Mistake | Why bad |
|---|---|
| Overlay rămâne pe ecran prea mult (>10 sec) | Distracting, viewer-ul se plictisește |
| Overlay dispare prea repede (<2 sec) | Patroni 50+ nu pot citi |
| Prea mult text per overlay (>10 cuvinte) | Cognitive overload — patroni renunță |
| Font script / Comic Sans | Amator, nu inspiră încredere |
| Background prea transparent (opacity <50%) | Text dispare pe screen recordings cu mult conținut |
| Animation prea cinematic (slide-burst, glitch) | Patroni RO 50+ percep ca "fake / scam" |
| Culori prea saturate (neon green, hot pink) | Out of brand HIR |
